Introduction

HTML is the standard markup language used to create web pages.

It gives structure to content so browsers can display text, images, links, and other elements.

<!DOCTYPE html>
  • HTML stands for HyperText Markup Language.

HTML Elements

HTML elements are the building blocks of a webpage.

Most elements have an opening tag, content, and a closing tag.

<p>This is a paragraph.</p>
  • Elements help organize page content.

Attributes

Attributes provide extra information about HTML elements.

They are usually written inside the opening tag of an element.

<a href="https://example.com">Link</a>
  • Attributes often use name and value pairs.

Forms

Forms allow users to submit information on a webpage.

Common form elements include inputs, labels, buttons, and text areas.

<input type="email"> <button>Submit</button>
  • Forms are useful for collecting user data.

Best Practices

Good HTML should be readable, organized, and semantic.

Using proper tags improves accessibility and search engine understanding.

<main>Page content</main>
  • Use semantic elements when possible.
